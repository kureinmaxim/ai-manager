Настройка YubiKey Аутентификации

Этот шаг необходим для основной, самой безопасной онлайн-аутентификации.

  1. Откройте ссылку в браузере: https://upgrade.yubico.com/getapikey/
  2. Введите ваш email, вставьте YubiKey и коснитесь его, когда попросят.
  3. Согласитесь с условиями и нажмите "Get API Key".
  4. Скопируйте полученные ключи:
    • Client ID: (число, например: 12345)
    • Secret Key: (строка, например: abcdefghijklmnop=)
  5. Вернитесь в приложение и вставьте эти ключи в форму настройки.

Это позволит входить в приложение без интернета. Настройте этот пароль на всех ваших ключах.

  1. Установите YubiKey Manager с официального сайта: yubico.com/support/download
  2. Откройте YubiKey Manager, вставьте ключ и перейдите в Applications -> OTP.
  3. Рекомендуется использовать второй слот. Нажмите Configure напротив Slot 2 (Long Touch).
  4. Выберите тип учетных данных: Static Password.
  5. Сгенерируйте пароль: Нажмите кнопку с тремя стрелками (⟳) для генерации случайного пароля. Рекомендуемая длина - 32+ символа.
  6. ВАЖНО: Скопируйте и надежно сохраните этот пароль. Он понадобится для файла `.env`.
  7. Нажмите Finish для записи на ключ.
  8. Добавьте сохраненный пароль в файл `.env` вашего проекта в переменную `YUBIKEY_STATIC_PASSWORDS`. Если паролей несколько, разделите их запятой.
